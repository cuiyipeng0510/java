## 数据结构与算法

### 1. 链表

#### 单向链表

```java
public Class Node{
	private int value;
	private Node next;
	public Node(int data){
		this.value = data; 
	}
}
```

双向链表

```java
public Class DoubleNode{
	private int value;
	private Node pre;
	private Node next;
	public DoubleNode(int data){
		this.value = data; 
	}
}
```
