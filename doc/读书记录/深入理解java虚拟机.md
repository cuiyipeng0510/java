###### 1. 方法区

1. 存放，类加载信息，常量，静态变量，及时编译代码缓存

3. 线程共享数据区域

###### 2. JVM Stack

1. 每个方法对应一个栈帧 Stack Frame
2. 存储局部变量表，操作数栈，动态链接，方法返回等信息
   - 局部变量表存放基础数据类型（boolean, byte, char, short, int, float, double, long）引用数据类型（reference 类型）
   - double, long 会占用两个变量槽，其余的数据类型只占用一个
